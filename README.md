# Authentifications
Bienvenue dans ce dépôt
Il regroupe mes différentes implémentations de systèmes d'authentification. L'objectif est de maîtriser les flux de connexion, la gestion des sessions.

## Projets actuels
  **Rest api**
  Une implémentation légère pour comprendre la logique fondamentale.
    - **Méthode:** Simulation de connexion de l'utilisateur par appel API
    - **Focus:** Gestion du State managemen, authentification de l'utilisateur et affichage des informations de l'utilisateur sur la page de profil
    **Lien** ["api rest"] 

  **Firebase**
  Intégration d'une solution Backend-as-a-Service (BaaS) robuste et sécurisée.    
    - **Technologie:** Firebase
    - **Méthode:** 
      - **Email / Mot de passe :** Création de compte et connexion classique.
      - **Authentification Anonyme :** Pour permettre une expérience utilisateur fluide avant l'inscription.
    - **Focus:** Persistence de l'authentification et gestion des erreurs Firebase.
